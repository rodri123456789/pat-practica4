# Práctica: Página Web Estática (TiendaPumillas)

Este repositorio contiene los archivos de la práctica de HTML y CSS. Se trata de una página web estática que simula el entorno de una tienda online de productos tecnológicos.

## Estructura del sitio

La web cumple con los requisitos de la práctica y consta de 4 páginas interconectadas:

* **Inicio (`index.html`):** Página principal con la presentación de la tienda.
* **Productos (`productos.html`):** Catálogo de productos que incluye contenido multimedia (imágenes).
* **Carrito (`carrito.html`):** Uso de tablas en HTML para mostrar el resumen de la compra y los precios.
* **Contacto (`contacto.html`):** Formulario web para recoger los datos de envío y método de pago del usuario.

## Tecnologías empleadas

* **HTML5:** Uso de etiquetas semánticas, estructuración de tablas, formularios y elementos multimedia.
* **CSS3:** Hoja de estilos externa (`style.css`) para mantener un diseño simple, uniforme y estructurado mediante el uso de contenedores y márgenes.
